# Introduction

Welcome to the **Vue Management System** documentation.

## Features

- **Dashboard**: Overview of your system status.
- **Spider**: Powerful crawling tools for Pixiv and NcMusic.
- **Resource Management**: 
  - **Image Library**: Browse and manage collected images with pagination and detail view.
  - **Useful Sites**: A collection of useful websites.
- **User Center**:
  - **Account**: Manage your profile and avatar.
  - **My Uploads**: View and manage your uploaded images.
  - **Collections**: Organize your favorite resources.
- **Settings**: Customizable user and system settings.

## Getting Started

Navigate through the sidebar to access different modules.
